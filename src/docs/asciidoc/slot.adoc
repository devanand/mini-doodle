== Slots

=== Create a slot

A slot is created with a start time and a duration; the end time is derived
by the server. Creation is rejected if the start time is in the past, if the
range is invalid, if the owner does not exist, or if the range overlaps an
existing slot for that owner.

==== Overlap conflict

Two slots for the same owner may not overlap. This is enforced both at
validation time (`NoOverlapRule`) and, since the validation check and the
insert are not atomic, again by a database exclusion constraint - see the
README's design notes for why both layers exist.

==== Bounds

The window may not exceed 100 days, and results are capped at 500 slots,
ordered ascending by start time. Both are stated in `400 Bad Request` errors
when violated. See the README for the reasoning and the cursor-pagination
upgrade path this leaves open.
