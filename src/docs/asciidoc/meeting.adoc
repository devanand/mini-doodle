== Meetings

A meeting books exactly one slot. Booking marks the slot BUSY; cancelling a
meeting deletes it and frees the slot again - both operations happen in one
transaction, so a meeting can never exist while its slot is FREE, and a slot
can never be BUSY without a corresponding meeting.

=== Book a meeting

The organizer is derived from the slot's owner, not supplied by the client.
Booking is rejected if the slot does not exist, or if it is already booked.
